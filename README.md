## 快速开始

### 本地开发
```bash
# 克隆仓库
git clone https://github.com/bianchunshan/ai-daily-v2.git

# 进入目录
cd ai-daily-v2

# 启动本地服务器
python3 -m http.server 8080

# 访问 http://localhost:8080
```

### 部署到GitHub Pages
1. Fork 本仓库
2. 进入 Settings → Pages
3. Source 选择 "Deploy from a branch"
4. Branch 选择 "main" / "/ (root)"
5. 点击 Save

## 数据架构

```
data/
├── news/                    # 新闻数据
│   ├── all.json            # 全部1100+条
│   ├── ai.json             # AI板块
│   ├── semiconductor.json  # 半导体板块
│   └── ...                 # 其他板块
├── stocks/                 # 股票数据库
│   └── database.json       # 150+股票信息
└── schema.json            # 数据模型定义
```
